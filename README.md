# HR Data Analyst Report
Welcome to the HR Data Analyst project repository. This project demonstrates end-to-end data analysis workflow, from database management to visualization, aimed at providing actionable insights into organizational attrition and employee distribution.

# Project Overview
This project analyzes HR data to identify key trends in employee turnover, attrition rates by department, and workforce demographics. The objective is to provide stakeholders with a clear understanding of the "who, where, and why" behind employee attrition to facilitate data-driven HR strategies.

# Key Insights
- Key Metrics: Employee Headcount, Attrition Rate, Average Salary, Tenure Distribution, Department Performance
- Attrition Patterns: Identified that the Operations department exhibits the highest attrition rate.
- Turnover Demographics: Discovered that senior specialists and specialists are the primary groups contributing to out-flow.
- Departmental Dynamics: While Operations has the highest percentage rate, the Sales department shows a high volume of total employee departures.
